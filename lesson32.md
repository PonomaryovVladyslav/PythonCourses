## –ò—Ç–µ—Ä–∞—Ç–æ—Ä—ã
–î–æ–∫—É–º–µ–Ω—Ç–∞—Ü–∏—è: https://docs.python.org/3/tutorial/classes.html#iterators


–í–æ –º–Ω–æ–≥–∏—Ö —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã—Ö —è–∑—ã–∫–∞—Ö –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è –∏—Å–ø–æ–ª—å–∑—É—é—Ç —Ç–∞–∫–∏–µ —Å—É—â–Ω–æ—Å—Ç–∏ –∫–∞–∫ –∏—Ç–µ—Ä–∞—Ç–æ—Ä—ã. –û—Å–Ω–æ–≤–Ω–æ–µ –∏—Ö –Ω–∞–∑–Ω–∞—á–µ–Ω–∏–µ ‚Äì —ç—Ç–æ
—É–ø—Ä–æ—â–µ–Ω–∏–µ –Ω–∞–≤–∏–≥–∞—Ü–∏–∏ –ø–æ —ç–ª–µ–º–µ–Ω—Ç–∞–º –æ–±—ä–µ–∫—Ç–∞, –∫–æ—Ç–æ—Ä—ã–π, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ, –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π –Ω–µ–∫–æ—Ç–æ—Ä—É—é –∫–æ–ª–ª–µ–∫—Ü–∏—é (—Å–ø–∏—Å–æ–∫, —Å–ª–æ–≤–∞—Ä—å
–∏ —Ç.–ø.). –Ø–∑—ã–∫ Python –≤ —ç—Ç–æ–º —Å–ª—É—á–∞–µ –Ω–µ –∏—Å–∫–ª—é—á–µ–Ω–∏–µ –∏ –≤ –Ω–µ–º —Ç–æ–∂–µ –µ—Å—Ç—å –ø–æ–¥–¥–µ—Ä–∂–∫–∞ –∏—Ç–µ—Ä–∞—Ç–æ—Ä–æ–≤. –ò—Ç–µ—Ä–∞—Ç–æ—Ä –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π
–æ–±—ä–µ–∫—Ç –ø–µ—Ä–µ—á–∏—Å–ª–∏—Ç–µ–ª—å, –∫–æ—Ç–æ—Ä—ã–π –¥–ª—è –¥–∞–Ω–Ω–æ–≥–æ –æ–±—ä–µ–∫—Ç–∞ –≤—ã–¥–∞–µ—Ç —Å–ª–µ–¥—É—é—â–∏–π —ç–ª–µ–º–µ–Ω—Ç –ª–∏–±–æ –±—Ä–æ—Å–∞–µ—Ç –∏—Å–∫–ª—é—á–µ–Ω–∏–µ, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤
–±–æ–ª—å—à–µ –Ω–µ—Ç.

–û—Å–Ω–æ–≤–Ω–æ–µ –º–µ—Å—Ç–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –∏—Ç–µ—Ä–∞—Ç–æ—Ä–æ–≤ ‚Äì —ç—Ç–æ —Ü–∏–∫–ª `for`. –ï—Å–ª–∏ –≤—ã –ø–µ—Ä–µ–±–∏—Ä–∞–µ—Ç–µ —ç–ª–µ–º–µ–Ω—Ç—ã –≤ –Ω–µ–∫–æ—Ç–æ—Ä–æ–º —Å–ø–∏—Å–∫–µ –∏–ª–∏ —Å–∏–º–≤–æ–ª—ã –≤
—Å—Ç—Ä–æ–∫–µ —Å –ø–æ–º–æ—â—å—é —Ü–∏–∫–ª–∞ `for`, —Ç–æ —Ñ–∞–∫—Ç–∏—á–µ—Å–∫–∏ —ç—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –ø—Ä–∏ –∫–∞–∂–¥–æ–π –∏—Ç–µ—Ä–∞—Ü–∏–∏ —Ü–∏–∫–ª–∞ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –æ–±—Ä–∞—â–µ–Ω–∏–µ –∫
–∏—Ç–µ—Ä–∞—Ç–æ—Ä—É, —Å–æ–¥–µ—Ä–∂–∞—â–µ–º—É—Å—è –≤ —Å—Ç—Ä–æ–∫–µ/—Å–ø–∏—Å–∫–µ —Å —Ç—Ä–µ–±–æ–≤–∞–Ω–∏–µ–º –≤—ã–¥–∞—Ç—å —Å–ª–µ–¥—É—é—â–∏–π —ç–ª–µ–º–µ–Ω—Ç, –µ—Å–ª–∏ —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤ –æ–±—ä–µ–∫—Ç–µ –±–æ–ª—å—à–µ –Ω–µ—Ç,
—Ç–æ –∏—Ç–µ—Ä–∞—Ç–æ—Ä –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç –∏—Å–∫–ª—é—á–µ–Ω–∏–µ, –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ–º–æ–µ –≤ —Ä–∞–º–∫–∞—Ö —Ü–∏–∫–ª–∞ `for` –Ω–µ–∑–∞–º–µ—Ç–Ω–æ –¥–ª—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è.

–ü—Ä–∏–≤–µ–¥–µ–º –Ω–µ—Å–∫–æ–ª—å–∫–æ –ø—Ä–∏–º–µ—Ä–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ –ø–æ–º–æ–≥—É—Ç –ª—É—á—à–µ –ø–æ–Ω—è—Ç—å —ç—Ç—É –∫–æ–Ω—Ü–µ–ø—Ü–∏—é. –î–ª—è –Ω–∞—á–∞–ª–∞ –≤—ã–≤–µ–¥–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –ø—Ä–æ–∏–∑–≤–æ–ª—å–Ω–æ–≥–æ
—Å–ø–∏—Å–∫–∞ –Ω–∞ —ç–∫—Ä–∞–Ω.

```python
num_list = [1, 2, 3, 4, 5]
for i in num_list:
    print(i)

# Output:

# 2
# 3
# 4
# 5
```

–ö–∞–∫ —É–∂–µ –±—ã–ª–æ —Å–∫–∞–∑–∞–Ω–æ, –æ–±—ä–µ–∫—Ç—ã, —ç–ª–µ–º–µ–Ω—Ç—ã –∫–æ—Ç–æ—Ä—ã—Ö –º–æ–∂–Ω–æ –ø–µ—Ä–µ–±–∏—Ä–∞—Ç—å –≤ —Ü–∏–∫–ª–µ `for`, —Å–æ–¥–µ—Ä–∂–∞—Ç –≤ —Å–µ–±–µ –æ–±—ä–µ–∫—Ç –∏—Ç–µ—Ä–∞—Ç–æ—Ä, –¥–ª—è
—Ç–æ–≥–æ, —á—Ç–æ–±—ã –µ–≥–æ –ø–æ–ª—É—á–∏—Ç—å, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Ñ—É–Ω–∫—Ü–∏—é `iter()`, –∞ –¥–ª—è –∏–∑–≤–ª–µ—á–µ–Ω–∏—è —Å–ª–µ–¥—É—é—â–µ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞ –∏–∑ –∏—Ç–µ—Ä–∞—Ç–æ—Ä–∞ ‚Äì
—Ñ—É–Ω–∫—Ü–∏—é `next()`.

```python

# StopIteration
```

–ö–∞–∫ –≤–∏–¥–Ω–æ –∏–∑ –ø—Ä–∏–≤–µ–¥–µ–Ω–Ω–æ–≥–æ –≤—ã—à–µ –ø—Ä–∏–º–µ—Ä–∞, –≤—ã–∑–æ–≤ —Ñ—É–Ω–∫—Ü–∏–∏ `next(itr)` –∫–∞–∂–¥—ã–π —Ä–∞–∑ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Å–ª–µ–¥—É—é—â–∏–π —ç–ª–µ–º–µ–Ω—Ç –∏–∑ —Å–ø–∏—Å–∫–∞, –∞
–∫–æ–≥–¥–∞ —ç—Ç–∏ —ç–ª–µ–º–µ–Ω—Ç—ã –∑–∞–∫–∞–Ω—á–∏–≤–∞—é—Ç—Å—è, –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç—Å—è –∏—Å–∫–ª—é—á–µ–Ω–∏–µ `StopIteration`.

### –ü–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –∏ –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–µ –æ–±—ä–µ–∫—Ç—ã

–ü–æ —Å—É—Ç–∏, –≤—Å—è —Ä–∞–∑–Ω–∏—Ü–∞ –º–µ–∂–¥—É –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—è–º–∏ –∏ –∏—Ç–µ—Ä–∏—Ä—É–µ–º—ã–º–∏ –æ–±—ä–µ–∫—Ç–∞–º–∏ (**–ù–µ –∏—Ç–µ—Ä–∞—Ç–æ—Ä–∞–º–∏**), –∑–∞–∫–ª—é—á–∞–µ—Ç—Å—è –≤ —Ç–æ–º, —á—Ç–æ –≤
–ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—è—Ö —ç–ª–µ–º–µ–Ω—Ç—ã —É–ø–æ—Ä—è–¥–æ—á–µ–Ω—ã.

–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç—è–º–∏ —è–≤–ª—è—é—Ç—Å—è —Å–ø–∏—Å–∫–∏, –∫–æ—Ä—Ç–µ–∂–∏ –∏ –¥–∞–∂–µ —Å—Ç—Ä–æ–∫–∏.

–ò—Ç–µ—Ä–∏—Ä—É–µ–º—ã–µ –æ–±—ä–µ–∫—Ç—ã –∂–µ, –Ω–∞–ø—Ä–æ—Ç–∏–≤, –Ω–µ —É–ø–æ—Ä—è–¥–æ—á–µ–Ω—ã, –Ω–æ, —Ç–µ–º –Ω–µ –º–µ–Ω–µ–µ, –º–æ–≥—É—Ç –±—ã—Ç—å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω—ã —Ç–∞–º, –≥–¥–µ —Ç—Ä–µ–±—É–µ—Ç—Å—è –∏—Ç–µ—Ä–∞—Ü–∏—è:
—Ü–∏–∫–ª `for`, –≤—ã—Ä–∞–∂–µ–Ω–∏—è-–≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä—ã, —Å–ø–∏—Å–∫–æ–≤—ã–µ –≤–∫–ª—é—á–µ–Ω–∏—è (list comprehensions) ‚Äî –∫–∞–∫ –ø—Ä–∏–º–µ—Ä—ã.


#### async for

`async for` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ã—Ö –∏—Ç–µ—Ä–∞—Ç–æ—Ä–æ–≤ ‚Äî –æ–±—ä–µ–∫—Ç–æ–≤ —Å –º–µ—Ç–æ–¥–∞–º–∏ `__aiter__` –∏ `__anext__`:

```python
    def __init__(self, stop):
        self.current = 0
        self.stop = stop

    def __aiter__(self):
        return self

        if self.current >= self.stop:
            raise StopAsyncIteration
        await asyncio.sleep(0.5)  # –ò–º–∏—Ç–∞—Ü–∏—è –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ–π –æ–ø–µ—Ä–∞—Ü–∏–∏
        self.current += 1
        return self.current

        print(number)

# 1
# 2
# 3
# 4
# 5
```

```python
import aiohttp
import asyncio


async def main():
    timeout = aiohttp.ClientTimeout(total=10)
    async with aiohttp.ClientSession(timeout=timeout) as session:
        async with session.get("https://httpbin.org/get") as resp:
            print(resp.status)
            data = await resp.json()
            print(data.get("url"))


asyncio.run(main())

# Output:
# 200
# https://httpbin.org/get
```

```python
import aiohttp
import asyncio

sem = asyncio.Semaphore(10)

async def fetch(session, url):
    async with sem:
        async with session.get(url, timeout=10) as r:
            return await r.text()

async def main():
    urls = ["https://httpbin.org/get"] * 50
    async with aiohttp.ClientSession() as session:
        results = await asyncio.gather(*(fetch(session, u) for u in urls))
        print(len(results))


asyncio.run(main())
```
